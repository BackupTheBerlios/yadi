YADI

Mit 'yadi' können jffs2- und squashfs-images erstellt werden. Die Datei muss ausführbar sein.

yadi ruft folgende Dateien auf:
prepare check get_cvs configure patch_cvs make_it flfs cat_image neutrino enigma neutrino_enigma squashfs_flfs squashfs_kernel squashfs_move_files squashfs_create_image

Die Konfiguration z.B. der Verzeichnisse kann in yadi.conf vorgenommen werden.

ALLIN1

Mit 'allin1.sh' können jffs2-images erstellt werden. Die Datei muss ausführbar sein.

Die Konfiguration z.B. der Verzeichnisse muss dortselbst vorgenommen werden.
